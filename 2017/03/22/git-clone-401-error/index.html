git clone 401 error

环境：CentOS 6.5
Git版本：1.7
问题：使用git clone时遇到了401错误
解决办法：目前找到的解决办法是升级git版本，从github中拉取最新版git源码，编译安装即可。
git编译安装参考：https://my.oschina.net/antsky/blog/514586
